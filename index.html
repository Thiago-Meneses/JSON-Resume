JSON JavaScript Object Notation


O JSON pode ser usado para

Enviar dados para um servidor

Receber dados de um servidor

  • Os dados sempre são recebidos do servidor em formato String;
  • Os dados sempre são enviados ao servidor em formato String;

  • Os dados sempre virão entre aspas simples;
Exemplo
'{"name":"Ana", "age":19, "city":"Contagem", "key": "value", ...}'

Sintaxe

  • A sintaxe do JSON é baseada na Object Notation do Javascript;
  • Por ser apenas texto, pode ser interpretado por outras linguagens, além do próprio Javascript.


  • OBJETOS do JavaScript podem ser escritos em JSON;
    • As Chaves {   } sempre irão conter objetos dentro de si

  • VETORES do JavaScript também podem ser escritos em JSON.
    • Os Colchetes [   ] sempre irão conter vetores dentro de si


  • As informações são representadas na forma "key": "value"
    • Cada key e value é envolvido por aspas duplas "   "
    • Cada dupla key-value é separada da próxima por vírgula
    • o último não terá virgula por não ser seguido por mais nenhum key/value pair.

VALORES INVÁLIDOS

  • Funções

    • Não é recomendado usar funções no JSON, pois elas perderão seu escopo
    • É necessário usar o eval() para convertê-las novamente em funções
    • Não é recomendado usar a função eval()

    • Caso seja necessário incluir funções dentro do JSON, elas deverão ser escritas no formato String
    • Após isso, é necessário converter a String em função novamente:
      •                                             
const text = ' { "name": "Michael", "age": "function () { return 30; }" } ';
const obj = JSON.parse(text);
obj.age = eval(" (" + obj.age + ") ");

  • Datas

    • Caso seja necessário incluir datas dentro do JSON, estas deverão ser escritas no formato String
    • Após isso é necessário converter a String novamente em data: 
      •

const obj = JSON.parse(text);

const text = ' { "name": "Michael", "birth": 1984-12-14 } '
const obj = JSON.parse(text, function (key, value) {

Funções Javascript

O próprio Javascript tem funções específicas para o JSON:


JSON.Parse()

  • Converte as Strings JSON que foram retornadas por um servidor qualquer
    • As Strings Serão convertidas em objetos do Javascript

  • Converte também os Arrays JSON que foram retornadas por um servidor qualquer
    • Os Arrays serão convertidos em vetores do Javascript


JSON.Stringify()

  • Converte Javascript Objects para que sejam enviados para um servidor como Strings JSON

    Exemplo:

                                    
const obj = {name: "John", age: 30};
const myJSON = JSON.stringify(obj);

*myJSON agora é uma String, pronta para ser enviada para o servidor

  • Também converte Javascript Arrays em Strings JSON

    • Exemplo:

                                    
const arr = ["John", "Michael", "Lucas", "Joe"];
const myJSON = JSON.stringify(arr)
*myJSON agora é um Array, pronto para ser enviado para o servidor

  • A função JSON.Stringify() transforma datas em Strings na conversão
  • A função JSON.Stringify() irá eliminar quaisquer funções de um Javascript Object, tanto a "key" como o "value"

String vs Object Literal

JSON String

'{"name": "John", "age": 30, "car":null}'


JSON Object Literal

{"name": "John", "age": 30, "car": null}
  • Object Literals são contidos entre chaves {   }
  • Object Literals contêm key-value pairs
    • keys e values são separados por vírgula
    • keys devem ser strings e values devem ser de um tipo JSON válido

  • Mistankenly JSON Object Literals are usually called JSON Objects, but JSON cannot be an object, The data can only be a JSON when it is in a String format.
  • When converted to a Javascript variable, it becomes a Javascript Object.

  • A forma mais comum é criar uma variável no formato String JSON e convertê-la em Javascript Object

    • ex:

                                        
myJSON = '{"name":"John", "age":30, "car":null}';	*String JSON criada
myObj = JSON.parse(myJSON);		*String JSON convetida em objeto
  • Também é possível criar um Javascript Object diretamente de um JSON Object Literal

    • ex:

                                        
myObj = {"name":"John", "age":30, "car":null};


Acessando valores dentro de um objeto

  • Os valores dentro de um objeto podem ser acessados usando dois tipos de notações:
    • PONTO FINAL .
      x = myObj.name;

    • COLCHETES [ ]
      x = myObj["name"];


Loops dentro de um Objeto

As propriedades de um objeto podem ser acessadas com um loop for .. in
Exemplo: 

                                    
const text = ' {"name": "John", "age": 30, "car": null} ';
const myObj = JSON.parse(text);
for ( const x in myObj ) {
    console.log (x);   		//Utilizando apenas a variável: mostrada a key
    console.log(myObj[x]) 	//Utilizando a variável entre colchetes: mostrada o value
}


JSON Array Literal

JSON String = '["Ford", "BMW", "Fiat"]'
Array Literal = ["Ford", "BMW", "Fiat"] *Está dentro da JSON String

  • Arrays dentro de Objects
    •                                     
const text =  '{ "name": "John", "age": 30, "cars": ["Ford", "BMW", "Fiat"] }'
const myObj = JSON,parse(text);

    Acessando os valores do array: 

                                        
myObj.cars[0]	//Usa-se colchetes e a enumeração do value desejado
  • Loop em um Array

    • Pode ser usado for ... in

                                        
const myJSON = '{"name":"John", "age":30, "cars":["Ford", "BMW", "Fiat"]}';
const myObj = JSON.parse(myJSON);
for ( let x in myObj.cars ){
    console.log( myObj.cars[x] );    //Usando-se a variável 'x' entre colchetes
}

    Tamém pode ser usado for

                                        
const myJSON = '{"name":"John", "age":30, "cars":["Ford", "BMW", "Fiat"]}'; 
const myObj = JSON.parse(myJSON); 
for ( let x=0; x<myObj.cars.length; x++){ 
    console.log(myObj.cars[x]); 
}
